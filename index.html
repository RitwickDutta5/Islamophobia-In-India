Islamophobia in India
Github

Muslims in India, constituting one of the largest Muslim populations globally, face discrimination, prejudice, and violence despite constitutional protections. Mob attacks by Hindu nationalist groups perpetuate communal riots, reflecting recurring sectarian violence.

Amid curtailed free speech and press freedom concerns, documenting pogroms and creating a resource base is crucial. It ensures accountability, justice, and historical preservation, countering marginalization.

This is a website that begins this documentation process, and will be a lifelong effort in making sure these stories are never forgotten.

To get a primer on Hindu-Muslim communal riots in India, you can find out more here

Click the buttons under each dropdown to examine major events since 1946. Each button corresponds to a marker that displays the location of these pogroms. The images under each button serve as a reference point.

Click the map pins to get a description of these events, and click the images under each button to learn more!
